# 🐦 Perch

A beautiful, lightweight menu bar to-do app for macOS. Stay productive without leaving your menu bar.

---

## ✨ Features

- **Quick Access** - Your tasks are always just a click away in the menu bar
- **Lightweight** - Built in Swift for optimal performance
- **Minimal Design** - Clean, distraction-free interface
- **Fast Notes** - Quickly add and manage your to-dos
- **Always Available** - Access your tasks from anywhere on your Mac

---

## 🛠️ Tech Stack

- **Language**: Swift (82.3%)
- **Build System**: Shell (17.7%)
- **Platform**: macOS

---

## 📋 Requirements

- macOS 12.0 or later
- Xcode 14.0 or later (for development)

---

## 🚀 Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/VladB-evs/Perch.git
cd Perch
```

2. Open in Xcode:
```bash
open Perch.xcodeproj
```

3. Build and run:
   - Select your target
   - Press `Cmd + R` to run

### Development

The project is structured with:
- Swift source files for the main application logic
- Shell scripts for build automation
